# Sistema Inteligente de Gerenciamento de Salas e Horários Acadêmicos

Projeto desenvolvido para a Residência Bolsa Futuro Digital.

## Instruções de Setup (Backend)
1. Abra o terminal na pasta do projeto.
2. Ative o ambiente virtual: `.\venv\Scripts\activate`
3. Instale as dependências (se necessário).
4. Inicie o servidor: `python manage.py runserver`
